The Code

                    
                        //get the start and end numbers from the inputs
//enrty point AKA controller function
function getValues() {

    let startValue = document.getElementById('startValue').value;
    let endValue = document.getElementById('endValue').value;

    let startNumber = parseInt(startValue);
    let endNumber = parseInt(endValue);

    let numberArray = generateNumbers(startNumber, endNumber);
    displayNumbers(numberArray);

}

//generate the range of numbers to display
//business logic function
function generateNumbers(start, end) {

    let basketOfNumbers = [];

    for (let number = start; number <= end; number = number + 1) {
        basketOfNumbers.push(number);
    }

    return basketOfNumbers; // => [0, 1, 2, ..., 100]

}

//showing the generated numbers on the page and bolding even numbers
//view function
function displayNumbers(numbers) {

    let results = '';

    for (let index = 0; index < numbers.length; index = index + 1) {
        let currentNumber = numbers[index];

        if (currentNumber % 2 == 0) {
            results = results + '<tr><td class="evenNumber">' + currentNumber + '</td></tr>';
        } else {
            results = results + '' + currentNumber + '';
        }

    }

    let tableBody = document.getElementById('results');
    tableBody.innerHTML = results;

}

site.js is a JavaScript file for "Counting Cruise" wich will generate a table of numbers within a specified range of 0 to 100, with even numbers highlighted in bold font.

Separation of concens allows us to be able to better understand and address any future bugs or changes.

The Code separated into 3 major functions

  1. Controller Function - getValues() is the function responsible to get the start and end numbers from the input as well as call any other function needed.
  2. Logic Function - generateNumbers() will need a starting and ending number to generate and return an array of numbers.
  3. View Function - displayNumbers() will take an array of numbers, formating and inserting each number into a table in index.html.

Now what happens when each function gets called?

getValues()
  • Will retrive a 'startValue' and 'endValue' from the document, formating them as a number and assigning them to 'startNumber' and 'endNumber'.
  • Will call generateNumbers() with 'startNumber' and 'endNumbers' as its input to generate an array of numbers called 'numberArray'.
  • Will call displayNumbers() with 'numberArray' as its input.
generateNumbers()
  • Will take two numbers ('start', 'end') to generate an array of numbers called 'basketOfNUmbers' from specified range using a for loop.
  • A for loop is made of four components an initialiser, condition, increaser/decreaser, and code to execute.
    In this case the code will initialize at 'start', increase the 'start' value by 1 each loop until the starting value is greater than or equal to the ending value 'end'.
  • Return 'basketOfNumbers'.
displayNumbers()
  • Will take an array of numbers called 'numbers'.
  • Use a for loop to generate a string 'results' made of each number formated as and html table row with even numbers highlighted in bold font.
  • The for loop will start at index = 0, increase index by 1 until it reahes the length of the 'numbers' array./li>
  • An if else statement is used, with a modulo operation used to identify even numbers. Adding even numbers a class of "evenNumber" to bold them.
  • A modulo operation (%) returns the remainder or signed remainder of a division, after one number is divided by another.
    In this if (number % 2) is equal to 0, it is an even number.
  • Will set the interHTML to 'results' a string witch has been formated to an HTML table with all the numbers inside the given range, with even numbers highlighted in bold font.
